# SecretSender 

SecretSender is a powerful web application built with React.js that allows users to send anonymous messages just like ngl


## Features

- Anonymous Message 
- Authentication 


## Tech Stack

- ReactJS
- Tailwind CSS
- Vercel
- Firebase
- Framer Motion 

## Demo

Check out the live demo [here](secretsenderweb.vercel.app/).
